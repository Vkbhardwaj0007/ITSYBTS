3 Zip File Upload करें

4 Zip File को Extract करें
  • File Manager में वापस जाएं
  • BTS_Website_Upload.zip पर right-click करें
  • Extract option select करें
  • Path confirm करें: /public_html/
  • Extract File(s) button पर click करें
