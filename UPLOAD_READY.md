## 🚀 Upload करने के लिए बस ये करें:

### आसान तरीका (5 minutes):

1. **cPanel में login करें**
   - अपने hosting provider की website पर जाएं
   - cPanel credentials use करें

2. **File Manager खोलें**
   - cPanel में "File Manager" icon पर click करें
   - `public_html` folder में जाएं

3. **Zip File Upload करें**
   - "Upload" button पर click करें
   - `BTS_Website_Upload.zip` select करें
   - Upload होने दें

4. **Extract करें**
   - Zip file पर right-click करें
   - "Extract" select करें
   - `/public_html/` में extract करें

5. **Done!** 🎉
   - अपना domain browser में खोलें
   - Website live होगी!
