TECHNICAL DATASHEET

Smart Home Automation Controller Version 3.1 (Relayless)

Project Overview

This datasheet defines the hardware specifications for the itsybts Home Automation Hub. The system is designed using a 100% Solid-State architecture, replacing mechanical relays with TRIACs for silent, arc-free, and high-frequency switching operations.

Core Features

  • Silent Operation: Zero clicking noise during switching.
  • Full Phase Control: Supports dimming on all 6 channels (4 Lights + 2 Fans).
  • Industrial Isolation: 2500V isolation between logic and high-voltage zones.
  • Advanced Safety: Individual fast-blow fuses and snubber circuits for every channel.
