## Image Optimization Tips

Before adding photos to the website:

1. **Resize** images to recommended dimensions
2. **Compress** using:
   - TinyPNG: https://tinypng.com
   - Squoosh: https://squoosh.app
3. **Target file size**:
   - Hero images: < 500KB each
   - Project images: < 300KB each
   - Team photo: < 400KB

## How to Replace Placeholder Images

1. Save your optimized photos in the `images/` folder
2. Open `index.html` in a text editor (Notepad, VS Code, etc.)
3. Find the Unsplash URLs (search for "unsplash.com")
4. Replace with your image paths:
   ```html
   <!-- OLD -->
   url('https://images.unsplash.com/photo-...')
   
   <!-- NEW -->
   url('images/your-photo.jpg')
   ```
5. Save the file
6. Refresh your browser

## Example Replacement

**Line 1200** (Hero Image 1):
```html
<!-- BEFORE -->
style="background-image: url('https://images.unsplash.com/photo-1518770660439-4636190af475?w=1920&h=1080&fit=crop');"

<!-- AFTER -->
style="background-image: url('images/hero-1.jpg');"
```
